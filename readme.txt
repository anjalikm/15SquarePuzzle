The .jar file is in directory "15SquarePuzzle-master".

The command line arguments are the difficulty level of the game. You can give any of the following:
1."EASY"
2."MEDIUM",
3."DIFFICULT"

e.g.
Go to directory "15SquarePuzzle".
>java -jar puzzle.jar "EASY"
